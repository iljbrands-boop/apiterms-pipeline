# Data license

Two different things live in this repo, under two different terms.

## Code — MIT

All Python and shell scripts (`ingest/`, `site/`) are MIT-licensed. See [LICENSE](LICENSE).

## Sample data — CC BY 4.0

The sample dataset in this repo — [`data/sample.jsonl`](data/sample.jsonl) (a dated 250-record
subset) and [`data/seed_domains.txt`](data/seed_domains.txt) (the coverage list) — is provided
under [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/). Use it
freely, including commercially; just attribute **API Terms (apiterms.com)**.

This is a **sample for evaluation**, not the full product. It is a point-in-time snapshot and
is not updated in this repo.

## Full dataset and change feed

The complete, continuously re-verified dataset — and the change history that powers the
[change feed](https://apiterms.com/changes/) — are the product, and are licensed separately:

- **Internal use** with attribution, and evaluation: free.
- **Redistribution, resale, or training** on the full dataset: licensed — see
  [apiterms.com/dataset](https://apiterms.com/dataset/).

The value of the full dataset is its freshness and its change history, neither of which can be
reconstructed from a static sample.
